about=
    Point-and-click AI segmentation tool that turns raster objects into vector polygons.
    KADAS Albireo 2 compatible version.

    • Fully automatic: AI model and dependencies install themselves

    • Works on any layer: local rasters (GeoTIFF, JPG, PNG) & online layers (WMS, XYZ tiles)

    • Precise: positive and negative click points control boundaries

    • Refinable: expand, contract, simplify, round corners, fill holes

    • Fast: Real-time segmentation & optimized for your hardware

    • Universal: Windows, macOS, Linux

    • Multilingual: English, French, Portuguese, Spanish, German, Italian

    • Compatible with KADAS Albireo 2

changelog=
    0.9.5 - Aligned with upstream QGIS AI-Segmentation v0.9.5
        - On-demand crop-based encoding (click-to-encode instead of full-raster pre-encoding)
        - SAM2 support (Python 3.10+) with SAM1 fallback
        - Online layer support (WMS, XYZ, WMTS)
        - Fast package installation via uv (Astral)
        - Refinement controls: expand, contract, simplify, smooth, fill holes
        - Frozen crop sessions for multi-area segmentation
        - Per-point undo with mask state history
        - Keyboard shortcuts (Space pan, Ctrl+Z, S, Enter, Esc)
        - GeoPackage export with layer groups
        - Plugin update check notifications
        - Improved error diagnostics (SSL, DLL, antivirus detection)

        - Added KADAS Albireo 2 interface support with custom ribbon tabs
        - Fixed WinError 1260: Corporate Group Policy compatibility
        - Added proxy/VPN support for enterprise networks
        - External cache directory for better isolation
        - Added Swiss locale support (it_CH, fr_CH, de_CH)
